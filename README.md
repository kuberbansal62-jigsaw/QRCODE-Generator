# QRCODE-Generator
# qR code generator in Python (Turtle)

A classic **qR code generator** built using Python's built-in **Turtle** graphics library. {aste the URL of any website or webpage and get an instant qR for the link



## 🛠️ Technologies Used

* Python 3.x
* Turtle Graphics
* Random module
* Time module
